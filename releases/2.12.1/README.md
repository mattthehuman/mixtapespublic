# v2.12.1 - GitHub Issues Integration

**Released: May 2026**

## What's new

### Feedback now goes to GitHub Issues

When you report a bug or share an idea through the app, your submission is now tracked as a GitHub issue rather than a Trello card.

- **Bug reports** - click the bug icon in the navigation bar
- **Feature ideas** - click the lightbulb icon in the navigation bar
- **Public tracker** - click the GitHub icon to browse open issues and follow progress

### Public issue tracker

Each submission is mirrored here in the public repository so the community can see what is being worked on. Only the description is shown publicly - submitter and site information is kept private.

### Automatic resolution

When a bug or feature is resolved in the private codebase, the corresponding public issue here is closed automatically with a note that it has been fixed.
