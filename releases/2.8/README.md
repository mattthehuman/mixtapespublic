# v2.8 - User Approval and Access Control

**Released: March 2026**

## What's new

- **User Approval Workflow** - Sites can now require admin approval before new sign-ups get access; users awaiting approval see a clear pending page
- **Access Denied Page** - Users blocked from signing up now see a friendly, informative page instead of a silent failure
- **Profile Picture Fix** - Google profile pictures now display correctly in the user panel
- **Bug Report URL** - Bug reports now automatically include the page URL where the issue occurred, making it easier to reproduce problems
