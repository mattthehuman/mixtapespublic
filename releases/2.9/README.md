# v2.9 - Spotify Listening

**Released: March 2026**

> **Note:** The Spotify listening feature introduced in this release was replaced by Last.fm integration in [v2.11](../2.11/) after Spotify removed Extended Web API access for independent developers.

## What's new

- **Spotify Listening Integration** - Connect your Spotify account to show friends what you are currently listening to on a mixtape
- **Listen History** - Tracks you have listened to on a released mixtape are highlighted so others can see what you have heard
- **Currently Listening Indicator** - A green border highlights your avatar on a track when you are actively listening to it
- **Privacy Consent** - A clear dialog explains exactly what Spotify data is used before you connect your account
- **Connect / Disconnect Spotify** - Link and unlink your Spotify account at any time; disconnecting removes all your listening history
- **Delete Button on Artwork** - The track delete button now appears as an overlay on the album artwork for a cleaner layout
- **Mobile Layout Improvements** - The user panel now fits correctly on small screens
