# v2.7 - Feedback System and Version Announcements

**Released: February 2026**

## What's new

- **Feedback System** - You can now report bugs and share ideas directly through the site
- **Bug Reports** - Click the bug icon next to the cat in the navigation bar to report an issue
- **Feature Ideas** - Click the lightbulb icon to suggest something new
- **Version Announcements** - The app now shows a one-time announcement when a new version is released, so you always know what has changed
- **Various Bug Fixes** - Mixtape pages now load correctly when refreshing directly on a URL; feedback modal displays correctly on mobile

> Bug and idea icons are only visible to signed-in users.
