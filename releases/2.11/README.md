# v2.11 - Last.fm Listening Integration

**Released: May 2026**

## What's new

After Spotify removed Extended Web API access for independent developers, listening integration has moved to Last.fm.

- **Connect Last.fm** - Click the Last.fm icon in the user panel and enter your Last.fm username
- **Spotify Scrobbling** - Spotify still scrobbles to Last.fm, so your listening activity is picked up automatically - no extra setup needed
- **Currently Listening** - A green border highlights your avatar on a track when you are actively listening to it
- **Listen History** - Friends can see which tracks you have heard on released mixtapes
- **Avatar Fallback** - Users without a profile picture now show their initial in listening avatars instead of a broken image
- **Connect / Disconnect** - Link and unlink your Last.fm account at any time from the user panel; disconnecting removes all your listening history
